# Wajeeha Marwa - Portfolio

A beautiful, modern portfolio website for Wajeeha Marwa, a Radiology Student at PEF College, Peshawar.

## Features

- 🎨 Modern glassmorphic design with gradient backgrounds
- 📱 Fully responsive layout (mobile, tablet, desktop)
- ✨ Smooth animations and transitions
- 🚀 Fast loading performance
- 📖 Clean semantic HTML structure

## Sections

- **About Me** - Introduction and professional overview
- **Education** - Timeline of academic achievements
- **Skills** - Radiology expertise (CT Scan, MRI, Ultrasound)
- **Contact** - Quick contact buttons

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript
- No external dependencies

## How to Deploy on GitHub Pages

1. Go to your GitHub repository: https://github.com/Huziafakhan/wajeeha
2. Click on **Settings** (top right)
3. Scroll down to **Pages** section in the left sidebar
4. Under **Source**, select:
   - Branch: **main**
   - Folder: **/(root)**
5. Click **Save**
6. Wait 1-2 minutes for GitHub to process
7. Your site will be live at: **https://huziafakhan.github.io/wajeeha/**

## Local Development

To view the portfolio locally:
1. Open `index.html` directly in your browser, OR
2. Use a local server:
   ```bash
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```
